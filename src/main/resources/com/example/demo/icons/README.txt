思维导图应用程序图标目录

请在此目录中放置以下图标文件:
1. new.png - 新建文件图标
2. open.png - 打开文件图标
3. save.png - 保存文件图标
4. save_as.png - 另存为图标
5. export.png - 导出图标
6. layout.png - 布局图标
7. theme.png - 主题图标
8. sidebar.png - 侧边栏切换图标
9. add_node.png - 添加节点图标
10. connect.png - 连接图标
11. edit.png - 编辑图标
12. delete.png - 删除图标
13. link.png - 连接模式图标
14. unlink.png - 退出连接图标

图标推荐尺寸: 16x16 或 24x24 像素
推荐格式: PNG格式，支持透明背景

如果无法获取上述图标，可以从以下来源获取免费图标:
- Material Design Icons: https://material.io/resources/icons/
- Font Awesome: https://fontawesome.com/icons
- Iconify: https://iconify.design/
